# ¿Cómo funciona?

- Arrastra bloques para construir algoritmos.
- Avanza por 50 niveles progresivos (variables, bucles, arrays, funciones, etc.).
- Recibe feedback inmediato y validación automática de tus soluciones.
- El progreso se guarda automáticamente en el navegador (localStorage).
- Laboratorio libre para experimentar sin restricciones.
