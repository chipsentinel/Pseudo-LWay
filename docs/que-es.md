# ¿Qué es Pseudo-LWay?

Pseudo-LWay es una plataforma web educativa para aprender y practicar pseudocódigo de forma visual y gamificada. Utiliza bloques tipo Blockly y un sistema de niveles curriculares para guiar el aprendizaje desde los fundamentos hasta problemas clásicos de algoritmia.

- Interfaz moderna y atractiva con temática neon.
- 50 niveles progresivos y laboratorio libre.
- Feedback inmediato y validación automática.
- Pensado para estudiantes, docentes y autodidactas.
